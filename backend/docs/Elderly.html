new Elderly()

Description:

  • Elderly user domain model Represents an elderly user who can request help and navigation assistance Extends the base User class with elderly-specific functionality like linking PINs

Source:
  • domain/elderly.js, line 5
Example
const elderly = new Elderly({
  userid: 'elderly-123',
  fullname: 'Jane Doe',
  email: 'jane@example.com',
  phone: '+6512345678',
  mobilityPreference: 'wheelchair',
  linkingPIN: '123456'
});
